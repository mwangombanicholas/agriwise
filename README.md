# AgriWise

AI-Powered Maize Disease Detection & Climate Advisory for Malawian Farmers

Built by **Mzuzu University Students** for the SAVE Project Innovation Call 2026.

---

## Features

- **96% accuracy** on validation data
- Detects: Healthy, Common Rust, Cercospora Leaf Spot, Northern Leaf Blight
- Treatment recommendations (chemical & organic)
- District-specific weather alerts

---

## Model Performance

| Metric | Value |
|--------|-------|
| Validation Accuracy | 95.87% |
| Training Images | 2,279 |

---

## Tech Stack

- **AI Model:** TensorFlow / Keras (CNN)
- **Frontend:** Streamlit
- **Language:** Python

---

## Installation

```bash
# Clone the repository
git clone https://github.com/mwangombanicholas/agriwise.git
cd agriwise

# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run agriwise_app.py
```
